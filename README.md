# TypeScript React Minesweeper Game

Classic game of Minesweeper recreated using React and TypeScript.

## Installation

1. Install dependencies in main project folder.

```
npm install
```

## Running the application

1. Run the React application

```
npm start
```

## Built With

- Typescript
- React
- VSCode
